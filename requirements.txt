### External Tools (Required for BLAST Notebook Only)

One notebook uses **NCBI BLAST+** for local protein alignment.

**Required executables:**

* `makeblastdb`
* `blastp`

**Installation**

macOS (Homebrew):

```bash
brew install blast
```

Ubuntu / Debian:

```bash
sudo apt-get update
sudo apt-get install ncbi-blast+
```

Windows:

* Download BLAST+ from NCBI
* Add the BLAST `bin` directory to your system `PATH`

**Verify installation:**

```bash
makeblastdb -version
blastp -version
```

> Note: If BLAST+ is not installed, only the BLAST notebook will fail. All other notebooks will run normally.

### Input Data Expectations

Notebooks expect standard bioinformatics file formats:

* DNA sequence strings
* FASTA files (`.fasta`, `.fa`, `.txt`)
* Gene expression matrices (`.txt`, `.tsv`, `.csv`)
* Genome annotation files (`.gff`, `.gff3`)
* BLAST output files (`.xml`)

Place all input files inside the `data/` directory and update file paths in notebooks accordingly.
